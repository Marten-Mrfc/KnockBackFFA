# Installation guide
:::danger[Platform Compatibility]
KnockBackFFA only works on **paper** servers. It will not work on Spigot or Bukkit servers.
:::

## Installing the KnockBackFFA plugin
To get the plugin correctly installed please follow the following steps:
1. Download the plugin via [BuiltByBit](https://builtbybit.com/resources/knockbackffa.47872/) and add it to your plugins folder.
2. Add our dependency [PlaceHolderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/) to your plugins folder.
3. Restart your server.

The plugin is now installed on your server.
